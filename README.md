# 绿动校园 - 碳积分小程序

基于UniApp开发的校园环保碳积分应用

## 功能特点

- **首页**: 用户积分展示、个人碳足迹卡片、每日减碳行动打卡入口
- **校园跑**: GPS定位跑步记录、运动轨迹可视化、跑步数据统计
- **低碳行为**: AI拍照识别垃圾分类、低碳行为打卡、积分奖励
- **积分商城**: 积分兑换、限时秒杀、碳汇资产认养
- **个人中心**: 碳减排数据统计、成就徽章、行为记录

## 技术栈

- UniApp (Vue 2)
- SCSS 样式
- 自定义图标字体

## 运行项目

```bash
# 安装依赖
npm install

# 运行到微信小程序
npm run dev:mp-weixin

# 运行到H5
npm run dev:h5

# 运行到App
npm run dev:app

# 构建微信小程序
npm run build:mp-weixin

# 构建H5
npm run build:h5
```

## 设计规范

### 主题色
- 主色: #10b981 (绿色)
- 辅色: #059669 (深绿)
- 强调色: #fbbf24 (金色)
- 背景色: #f8fafc (浅灰)

### 圆角规范
- 大卡片: 48rpx
- 中卡片: 32rpx
- 小卡片: 24rpx
- 按钮: 50rpx (圆形)

### 字体大小
- 大标题: 72rpx
- 页面标题: 44rpx
- 模块标题: 32rpx
- 正文: 28rpx
- 辅助文字: 22rpx
- 小标签: 20rpx

## 注意事项

1. 图标使用 emoji 作为占位符，正式项目请替换为真实图标
2. 图片使用网络图片，正式项目请替换为本地资源
3. TabBar 图标使用 SVG 格式，微信小程序需转换为 base64 或使用 iconPath
4. 建议使用 uni-ui 组件库增强交互效果
